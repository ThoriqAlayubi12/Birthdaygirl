# Selamat ulang tahun 🎉

Website HTML interaktif yang bisa:
- dibuka langsung sebagai file `index.html` di ponsel/laptop, atau
- dipublikasikan melalui GitHub Pages (`github.io`).

## Fitur
- Judul: **Selamat ulang tahun**
- Dinding ucapan berbentuk **box testimoni** yang scrollable.
- Checklist wishlist & doa yang bisa dicentang, dengan progres otomatis.
- Status checklist tersimpan di browser (localStorage).

## Cara pakai lokal (langsung kirim file HTML)
1. Kirim file `index.html` ke ponsel (WhatsApp/Telegram/email/AirDrop).
2. Buka file tersebut di browser mobile.

## Cara publish ke GitHub Pages
1. Buat repo GitHub (atau pakai repo ini) dan push semua file.
2. Masuk ke **Settings** repo.
3. Pilih **Pages**.
4. Pada **Build and deployment**:
   - Source: `Deploy from a branch`
   - Branch: `main` (root)
5. Simpan, tunggu proses deploy selesai.
6. Website tersedia di:
   `https://<username>.github.io/<nama-repo>/`

## Struktur file
- `index.html` — halaman utama website.
